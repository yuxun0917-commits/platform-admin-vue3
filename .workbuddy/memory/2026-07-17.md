## ⚠️ 提交前必须 git add 暂存修复（IDE 会 stash 未暂存改动）
- **现象**：vue-tsc 改完本地 exit 0，但用户 IDE 提交 hook 反复报"同一批原始错误"。根因：该 IDE 提交前把**未暂存的改动 stash 掉**再跑 `vue-tsc`，只改磁盘没 `git add` 的修复被 stash 带走，提交跑在还原后的旧代码上（components.d.ts 因上一轮已 `git add` 暂存故不报错，其余未暂存的全部回退）。
